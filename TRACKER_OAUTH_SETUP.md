# Получение OAuth токена для Yandex Tracker

## Проблема
IAM токен работает для Yandex Cloud API, но Yandex Tracker API требует OAuth токен пользователя.

## Решение: Получить OAuth токен

### Способ 1: Через веб-интерфейс Tracker (Рекомендуется)

1. Откройте https://tracker.yandex.ru/
2. Войдите в организацию **Regulatrix** (ID: bpfrsn8gjmg4f3b8ffa2)
3. Перейдите в **Настройки → Токены**: https://tracker.yandex.ru/settings/tokens
4. Нажмите **"Получить токен"**
5. Разрешите доступ приложению
6. Скопируйте полученный OAuth токен

### Способ 2: Создание приложения в Яндекс OAuth (для продвинутых)

1. Перейдите на https://oauth.yandex.ru/
2. Создайте новое приложение
3. Выберите необходимые права для Tracker API
4. Получите Client ID
5. Используйте для авторизации

**Проще использовать Способ 1** - токен из настроек Tracker

### Обновление .env

После получения OAuth токена:

```bash
YANDEX_TRACKER_TOKEN=ваш_oauth_токен_здесь
```

### Создание очереди VOICEBOT

После обновления токена выполните:

```bash
python scripts/check_or_create_queue.py
```

Или создайте очередь вручную:
1. Откройте https://tracker.yandex.ru/
2. Нажмите **"Создать очередь"**
3. Введите ключ: **VOICEBOT**
4. Название: **Voice Calendar Bot**
5. Шаблон: **Разработка**

### Создание задач

После создания очереди:

```bash
python scripts/create_tracker_tasks.py
```

## Альтернатива: Работа без Tracker

Проект может работать и без Яндекс Трекера. Отслеживание задач ведется через:
- TodoWrite в Claude Code
- Локальный файл TODO.md
- Git commits

Для работы бота Tracker не обязателен - это опциональная функция для автоматизации управления задачами.
